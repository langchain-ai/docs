The high-level, prebuilt components are organized into several packages, each with a specific focus:

| Package                                    | Description                                                                              | Installation                            |
| ------------------------------------------ | ---------------------------------------------------------------------------------------- | --------------------------------------- |
| `langgraph-prebuilt` (part of `langgraph`) | Prebuilt components to [**create agents**](/oss/quickstart)                                  | `pip install -U langgraph langchain`    |
| `langgraph-supervisor`                     | Tools for building [**supervisor**](/oss/multi-agent#supervisor) agents                  | `pip install -U langgraph-supervisor`   |
| `langgraph-swarm`                          | Tools for building a [**swarm**](/oss/multi-agent#swarm) multi-agent system              | `pip install -U langgraph-swarm`        |
| `langchain-mcp-adapters`                   | Interfaces to [**MCP servers**](/oss/mcp) for tool and resource integration              | `pip install -U langchain-mcp-adapters` |
| `langmem`                                  | Agent memory management: [**short-term and long-term**](/oss/add-memory) | `pip install -U langmem`                |
| `agentevals`                               | Utilities to [**evaluate agent performance**](/oss/evals)                                | `pip install -U agentevals`             |
