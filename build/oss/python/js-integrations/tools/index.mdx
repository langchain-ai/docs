---
title: Tools and Toolkits
---

[Tools](/oss/python/concepts/tools) are utilities designed to be called by a model: their inputs are designed to be generated by models, and their outputs are designed to be passed back to models.

A [toolkit](/oss/python/concepts/tools/#toolkits) is a collection of tools meant to be used together. For a list of toolkit integrations, see [this page](/oss/python/integrations/toolkits/).

<Info>
If you'd like to write your own tool, see [this how-to](/oss/python/how-to/custom_tools/). If you'd like to contribute an integration, see [Contributing integrations](/docs/contributing).
</Info>
