This page contains guides around resolving common errors you may find while building with LangGraph.
Errors referenced below will have an `lc_error_code` property corresponding to one of the below codes when they are thrown in code.

* [GRAPH_RECURSION_LIMIT](/oss/javascript/GRAPH_RECURSION_LIMIT)
* [INVALID_CONCURRENT_GRAPH_UPDATE](/oss/javascript/INVALID_CONCURRENT_GRAPH_UPDATE)
* [INVALID_GRAPH_NODE_RETURN_VALUE](/oss/javascript/INVALID_GRAPH_NODE_RETURN_VALUE)
* [MULTIPLE_SUBGRAPHS](/oss/javascript/MULTIPLE_SUBGRAPHS)
* [INVALID_CHAT_HISTORY](/oss/javascript/INVALID_CHAT_HISTORY)
