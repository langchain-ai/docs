# LangGraph.js setup

Learn how to set up your LangGraph Platform project using LangGraph.js.

## Basic Setup

Create a new Node.js project:

```bash
npm init -y
```

Install LangGraph.js:

```bash
npm install @langgraph/platform
```

## Configuration

Create a `langgraph.config.js` file:

```javascript
module.exports = {
  // Your LangGraph.js configuration
  server: {
    port: 3000
  },
  // Add more configuration options
};
```
