# Pyproject.toml setup

Learn how to set up your LangGraph Platform project using a `pyproject.toml` file.

## Basic Setup

Create a `pyproject.toml` file in your project root:

```toml
[project]
name = "my-langgraph-app"
version = "0.1.0"
description = "My LangGraph Platform Application"
requires-python = ">=3.9"

[tool.poetry.dependencies]
langgraph-platform = "^1.0.0"
python-dotenv = "^1.0.0"
```

## Installation

Install the dependencies using Poetry:

```bash
poetry install
```
