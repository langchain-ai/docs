## Installation and Setup

Install the Perplexity x LangChain integration package:

```
pip install langchain-perplexity
```

Get your API key from [here](https://docs.perplexity.ai/oss/getting-started).

## Chat models

See a variety of usage examples [here](/oss/integrations/chat/perplexity).

from langchain_perplexity import ChatPerplexity
