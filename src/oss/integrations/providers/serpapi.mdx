This page covers how to use the SerpAPI search APIs within LangChain. It is broken into two parts: installation and setup, and then references to the specific SerpAPI wrapper.

## Installation and Setup

* Install requirements with `pip install google-search-results`
* Get a SerpAPI api key and either set it as an environment variable (`SERPAPI_API_KEY`)

## Wrappers

### Utility

There exists a SerpAPI utility which wraps this API. To import this utility:

```
from langchain_community.utilities import SerpAPIWrapper
```

**API Reference:**[SerpAPIWrapper](https://python.langchain.com/api_reference/community/utilities/langchain_community.utilities.serpapi.SerpAPIWrapper.html)

For a more detailed walkthrough of this wrapper, see [this notebook](/src/oss/integrations/tools/serpapi).

### Tool

You can also easily load this wrapper as a Tool (to use with an Agent). You can do this with:

```
from langchain_community.agent_toolkits.load_tools import load_toolstools = load_tools(["serpapi"])
```

**API Reference:**[load\_tools](https://python.langchain.com/api_reference/community/agent_toolkits/langchain_community.agent_toolkits.load_tools.load_tools.html)

For more information on this, see [this page](/src/oss/how_to/tools_builtin)
