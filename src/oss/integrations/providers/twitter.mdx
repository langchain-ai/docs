> [Twitter](https://twitter.com/) is an online social media and social networking service.

## Installation and Setup

```
pip install tweepy
```

We must initialize the loader with the `Twitter API` token, and we need to set up the Twitter `username`.

## Document Loader

See a [usage example](/src/oss/integrations/document_loaders/twitter).

```
from langchain_community.document_loaders import TwitterTweetLoader
```

**API Reference:**[TwitterTweetLoader](https://python.langchain.com/api_reference/community/document_loaders/langchain_community.document_loaders.twitter.TwitterTweetLoader.html)
