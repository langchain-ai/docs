## Chat models

See the [installation instructions](/oss/integrations/chat/yuan2/#setting-up-your-api-server).

Yuan2.0 provided an OpenAI compatible API, and ChatYuan2 is integrated into langchain by using `OpenAI client`. Therefore, ensure the `openai` package is installed.

```
pip install openai
```

See a [usage example](/oss/integrations/chat/yuan2).

```
from langchain_community.chat_models import ChatYuan2
```

**API Reference:**[ChatYuan2](https://python.langchain.com/api_reference/community/chat_models/langchain_community.chat_models.yuan2.ChatYuan2.html)
