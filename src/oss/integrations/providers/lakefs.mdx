> [lakeFS](https://docs.lakefs.io/) provides scalable version control over the data lake, and uses Git-like semantics to create and access those versions.

## Installation and Setup

Get the `ENDPOINT`, `LAKEFS_ACCESS_KEY`, and `LAKEFS_SECRET_KEY`. You can find installation instructions [here](https://docs.lakefs.io/quickstart/launch.html).

## Document Loader

See a [usage example](/oss/integrations/document_loaders/lakefs).

```
from langchain_community.document_loaders import LakeFSLoader
```
