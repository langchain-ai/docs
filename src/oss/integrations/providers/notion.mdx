> [Notion](https://www.notion.so/) is a collaboration platform with modified Markdown support that integrates kanban boards, tasks, wikis and databases. It is an all-in-one workspace for notetaking, knowledge and data management, and project and task management.

## Installation and Setup

All instructions are in examples below.

## Document Loader

We have two different loaders: `NotionDirectoryLoader` and `NotionDBLoader`.

See [usage examples here](/src/oss/integrations/document_loaders/notion).

```
from langchain_community.document_loaders import NotionDirectoryLoader, NotionDBLoader
```

**API Reference:**[NotionDirectoryLoader](https://python.langchain.com/api_reference/community/document_loaders/langchain_community.document_loaders.notion.NotionDirectoryLoader.html) | [NotionDBLoader](https://python.langchain.com/api_reference/community/document_loaders/langchain_community.document_loaders.notiondb.NotionDBLoader.html)
