> [Nomic Atlas](https://docs.nomic.ai/index.html) is a platform for interacting with both small and internet scale unstructured datasets.

## Installation and Setup

* Install the Python package with `pip install nomic`
* `Nomic` is also included in langchains poetry extras `poetry install -E all`

## VectorStore

See a [usage example](/oss/integrations/vectorstores/atlas).

```
from langchain_community.vectorstores import AtlasDB
```

**API Reference:**[AtlasDB](https://python.langchain.com/api_reference/community/vectorstores/langchain_community.vectorstores.atlas.AtlasDB.html)
