> [Bagel](https://www.bagel.net/) (`Open Vector Database for AI`), is like GitHub for AI data. It is a collaborative platform where users can create, share, and manage vector datasets. It can support private projects for independent developers, internal collaborations for enterprises, and public contributions for data DAOs.

## Installation and Setup

```
pip install bagelML
```

## VectorStore
