> [KoNLPy](https://konlpy.org/) is a Python package for natural language processing (NLP) of the Korean language.

## Installation and Setup

You need to install the `konlpy` python package.

```
pip install konlpy
```

## Text splitter

See a [usage example](/src/oss/how_to/split_by_token/#konlpy).

```
from langchain_text_splitters import KonlpyTextSplitter
```

**API Reference:**[KonlpyTextSplitter](https://python.langchain.com/api_reference/text_splitters/konlpy/langchain_text_splitters.konlpy.KonlpyTextSplitter.html)
