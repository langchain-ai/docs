> [Context](https://context.ai/) provides user analytics for LLM-powered products and features.

## Installation and Setup

We need to install the `context-python` Python package:

```
pip install context-python
```

## Callbacks

See a [usage example](/oss/integrations/callbacks/context).

```
from langchain.callbacks import ContextCallbackHandler
```

**API Reference:**[ContextCallbackHandler](https://python.langchain.com/api_reference/community/callbacks/langchain_community.callbacks.context_callback.ContextCallbackHandler.html)
