> [OpenWeatherMap](https://openweathermap.org/) is an open-source weather service provider.

## Installation and Setup

```
pip install pyowm
```

We must set up the `OpenWeatherMap API token`.

## Document Loader

See a [usage example](/src/oss/integrations/document_loaders/weather).

```
from langchain_community.document_loaders import WeatherDataLoader
```

**API Reference:**[WeatherDataLoader](https://python.langchain.com/api_reference/community/document_loaders/langchain_community.document_loaders.weather.WeatherDataLoader.html)
