> [Cerebrium](https://docs.cerebrium.ai/cerebrium/getting-started/introduction) is a serverless GPU infrastructure provider. It provides API access to several LLM models.

See the examples in the [CerebriumAI documentation](https://docs.cerebrium.ai/examples/langchain).

## Installation and Setup

* Install a python package:

```
pip install cerebrium
```

* [Get an CerebriumAI api key](https://docs.cerebrium.ai/cerebrium/getting-started/installation) and set it as an environment variable (`CEREBRIUMAI_API_KEY`)

## LLMs

See a [usage example](/oss/integrations/llms/cerebriumai).

```
from langchain_community.llms import CerebriumAI
```

**API Reference:**[CerebriumAI](https://python.langchain.com/api_reference/community/llms/langchain_community.llms.cerebriumai.CerebriumAI.html)
