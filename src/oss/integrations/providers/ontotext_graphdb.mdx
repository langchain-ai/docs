> [Ontotext GraphDB](https://graphdb.ontotext.com/) is a graph database and knowledge discovery tool compliant with RDF and SPARQL.

## Dependencies

Install the [rdflib](https://github.com/RDFLib/rdflib) package with

```
pip install rdflib==7.0.0
```

## Graph QA Chain

Connect your GraphDB Database with a chat model to get insights on your data.

See the notebook example [here](/oss/integrations/graphs/ontotext).

```
from langchain_community.graphs import OntotextGraphDBGraphfrom langchain.chains import OntotextGraphDBQAChain
```

**API Reference:**[OntotextGraphDBGraph](https://python.langchain.com/api_reference/community/graphs/langchain_community.graphs.ontotext_graphdb_graph.OntotextGraphDBGraph.html) | [OntotextGraphDBQAChain](https://python.langchain.com/api_reference/community/chains/langchain_community.chains.graph_qa.ontotext_graphdb.OntotextGraphDBQAChain.html)
