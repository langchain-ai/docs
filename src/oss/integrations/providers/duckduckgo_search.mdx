> [DuckDuckGo Search](https://github.com/deedy5/duckduckgo_search) is a package that searches for words, documents, images, videos, news, maps and text translation using the `DuckDuckGo.com` search engine. It is downloading files and images to a local hard drive.

## Installation and Setup

You have to install a python package:

```
pip install duckduckgo-search
```

## Tools

See a [usage example](/src/oss/integrations/tools/ddg).

There are two tools available:

```
from langchain_community.tools import DuckDuckGoSearchRunfrom langchain_community.tools import DuckDuckGoSearchResults
```

**API Reference:**[DuckDuckGoSearchRun](https://python.langchain.com/api_reference/community/tools/langchain_community.tools.ddg_search.tool.DuckDuckGoSearchRun.html) | [DuckDuckGoSearchResults](https://python.langchain.com/api_reference/community/tools/langchain_community.tools.ddg_search.tool.DuckDuckGoSearchResults.html)
