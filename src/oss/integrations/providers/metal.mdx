This page covers how to use [Metal](https://getmetal.io) within LangChain.

## What is Metal?

Metal is a managed retrieval & memory platform built for production. Easily index your data into `Metal` and run semantic search and retrieval on it.

![Screenshot of the Metal dashboard showing the Browse Index feature with sample data.](/images/oss/integrations/providers/metal/assets/images/MetalDash-6bcdc3c36355ca99a18e4ced92789b3e.png "Metal Dashboard Interface")

## Quick start

Get started by [creating a Metal account](https://app.getmetal.io/signup).

Then, you can easily take advantage of the `MetalRetriever` class to start retrieving your data for semantic search, prompting context, etc. This class takes a `Metal` instance and a dictionary of parameters to pass to the Metal API.

```
from langchain.retrievers import MetalRetrieverfrom metal_sdk.metal import Metalmetal = Metal("API_KEY", "CLIENT_ID", "INDEX_ID");retriever = MetalRetriever(metal, params={"limit": 2})docs = retriever.invoke("search term")
```

**API Reference:**[MetalRetriever](https://python.langchain.com/api_reference/community/retrievers/langchain_community.retrievers.metal.MetalRetriever.html)
