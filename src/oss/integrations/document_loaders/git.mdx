## Clone repository from url

```
from langchain_community.document_loaders import GitLoader
```

**API Reference:**[GitLoader](https://python.langchain.com/api_reference/community/document_loaders/langchain_community.document_loaders.git.GitLoader.html)

```
loader = GitLoader(    clone_url="https://github.com/langchain-ai/langchain",    repo_path="./example_data/test_repo2/",    branch="master",)
```

## Filtering files to load

```
from langchain_community.document_loaders import GitLoader# e.g. loading only python filesloader = GitLoader(    repo_path="./example_data/test_repo1/",    file_filter=lambda file_path: file_path.endswith(".py"),)
```
