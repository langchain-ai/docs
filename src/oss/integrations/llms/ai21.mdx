  # AI21 Contextual Answer

  You can use AI21's contextual answers model to receives text or document, serving as a context, and a question and returns an answer based entirely on this context.

  This means that if the answer to your question is not in the document, the model will indicate it (instead of providing a false answer)

  ```
  from langchain_ai21 import AI21ContextualAnswerstsm = AI21ContextualAnswers()response = tsm.invoke(input={"context": "Your context", "question": "Your question"})
  ```

  You can also use it with chains and output parsers and vector DBs

  ```
  from langchain_ai21 import AI21ContextualAnswersfrom langchain_core.output_parsers import StrOutputParsertsm = AI21ContextualAnswers()chain = tsm | StrOutputParser()response = chain.invoke(    {"context": "Your context", "question": "Your question"},)
  ```

  **API Reference:**[StrOutputParser](https://python.langchain.com/api_reference/core/output_parsers/langchain_core.output_parsers.string.StrOutputParser.html)
