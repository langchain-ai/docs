# Understanding LangSmith

LangSmith is a platform for debugging, testing, evaluating, and monitoring your LLM applications. It helps you develop and deploy reliable, high-quality applications by providing tools for:

- Tracing and debugging LLM applications
- Testing and evaluating model outputs
- Monitoring production performance
- Managing and versioning prompts
- Collecting and analyzing feedback

## Key features

- **Tracing**: Track the execution of your LLM applications in real-time
- **Evaluation**: Test and evaluate your models with custom metrics
- **Monitoring**: Monitor your application's performance in production
- **Prompt Management**: Version control and manage your prompts
- **Collaboration**: Work together with your team on prompt engineering
