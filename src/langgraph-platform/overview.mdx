# Overview

LangGraph Platform is a commercial framework for deploying LLM applications. It provides a comprehensive set of tools and features to help you build, deploy, and manage your LLM applications in production.

## Key Features

- Production-ready orchestration
- Built-in authentication and access control
- Assistant and thread management
- Real-time streaming capabilities
- Human-in-the-loop workflows
- Advanced debugging tools
- Flexible deployment options

## Getting Started

To get started with LangGraph Platform, check out our [Quickstart Guide](/langgraph-platform/quickstart) or explore the [Application Setup](/langgraph-platform/app-setup/requirements-txt) section to learn about the different ways to set up your application.
