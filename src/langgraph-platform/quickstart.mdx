# Quickstart guide

This guide will help you get started with LangGraph Platform quickly. Follow these steps to set up your first application.

## Prerequisites

- Python 3.9 or higher
- pip (Python package manager)
- A code editor of your choice

## Installation

1. Create a new Python project and set up a virtual environment:

```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

2. Install LangGraph Platform:

```bash
pip install langgraph-platform
```

## Basic Setup

1. Create a new file `app.py`:

```python
from langgraph_platform import LangGraphPlatform

app = LangGraphPlatform()

@app.assistant
def my_assistant():
    return "Hello from LangGraph Platform!"

if __name__ == "__main__":
    app.run()
```

2. Run your application:

```bash
python app.py
```
