# CLI tools

The LangGraph Platform CLI provides command-line tools for managing and deploying your applications.

## Installation

Install the CLI globally:

```bash
pip install langgraph-platform-cli
```

## Basic Commands

```bash
# Start the server
langgraph serve

# Deploy your application
langgraph deploy

# Monitor your application
langgraph monitor

# Manage assistants
langgraph assistants list
langgraph assistants create
```

## Configuration

Configure the CLI using environment variables or a config file:

```bash
# Using environment variables
export LANGRAPH_API_KEY=your_api_key
export LANGRAPH_ENV=production

# Or using config file
langgraph config set api_key your_api_key
langgraph config set env production
```
