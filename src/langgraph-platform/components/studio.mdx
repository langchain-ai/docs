# LangGraph studio

LangGraph Studio is a web-based interface for monitoring and managing your LangGraph Platform applications.

## Features

- Real-time monitoring
- Workflow visualization
- Debugging tools
- Assistant management
- Thread management
- Performance analytics

## Accessing Studio

Access Studio through your web browser:

```bash
# Start Studio locally
langgraph studio

# Or access the cloud version
https://studio.langgraph.ai
```

## Key Features

1. **Dashboard**
   - Real-time metrics
   - System health
   - Active workflows

2. **Workflow Editor**
   - Visual workflow builder
   - Node configuration
   - Edge management

3. **Debug Tools**
   - Breakpoint management
   - Time travel debugging
   - State inspection
